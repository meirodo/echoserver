# エコーサーバー

クライアントから受け取ったデータをそのまま送り返す（エコーバックする）サーバー。

## 実行方法

1. サーバーを起動
    ```bash
    $ ./build/es_server
    ```
1. クライアントを起動
    ```bash
    $ ./build/es_client
    ```

## 終了方法

1. `Ctrl+C`でクライアントを終了
1. `Ctrl+C`でサーバーを終了

## 注意

サーバーに同時に接続できるのは1台のみ。すでに接続されたクライアントがある場合、そのクライアントを終了しないと他のクライアントはサーバーに接続できません。
